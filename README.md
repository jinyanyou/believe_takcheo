# 탁처의 갸루침

탁처를 믿습니까? 오. 그대는 믿어야하는.
탁처님와 함께 오늘의 갸루침, 고민 상담, 경전, 만다라 인터랙션을 제공합니다. 야-루🙏

## 배포 주소

https://jinyanyou.github.io/believe_takcheo/

## 주요 기능

- 오늘의 갸루침 랜덤 표시
- 고민 입력에 따른 탁처식 답변 생성
- 장과 절 형태의 짧은 경전 목록
- 주제를 누르면 문구가 바뀌는 만다라 인터랙션
- 데스크탑과 모바일에 대응하는 반응형 레이아웃

## 배포

이 프로젝트는 별도 빌드 과정이 없는 HTML, CSS, JavaScript 정적 사이트입니다. GitHub Pages에서 `main` 브랜치의 `/ (root)`를 배포 소스로 설정하면 됩니다.
